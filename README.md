# Task Manager REST API

## Description
A RESTful API that allows users to create, read, update, and delete tasks using server-side logic.

## Tech Stack
- Node.js
- Express
- MongoDB

## Features
- CRUD operations for tasks
- REST API architecture
- Input validation
- Modular backend structure

## Setup
1. Clone the repository
2. Install dependencies using `npm install`
3. Create a `.env` file using `.env.example`
4. Start the server using `npm start`

## Notes
This project was developed as part of a learning-based full stack development experience.
All sensitive credentials and environment variables have been excluded.
